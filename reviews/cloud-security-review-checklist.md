# Cloud Security Review Checklist

## Identity

- Who can access the platform
- Which roles are high privilege
- Are service accounts scoped correctly

## Network

- Which workloads are public
- Which workloads should be private
- Are segmentation boundaries defined

## Data protection

- Encryption requirements met
- Sensitive services protected
- Logging and monitoring enabled

## Governance

- Guardrails defined
- Auditability available
- High-risk changes reviewed
